## Project: Sutrah.co
Website service business for Malaysian SMEs. Build + monthly retainer model.
Live at: https://sutrah-co.vercel.app
GitHub: https://github.com/nerolynx27/sutrah-co

## Stack
- Next.js 14 (App Router, Turbopack) + Tailwind CSS v4
- Vercel (hosting) + Supabase (database) + Resend (email)
- Design: dark (#0D0D0D) + gold (#BF932A) palette, Plus Jakarta Sans
